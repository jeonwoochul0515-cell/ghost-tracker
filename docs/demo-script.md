# Demo Script

## 1분 데모 (의사결정자용)

1. (00:00) **메인 대시보드 열기** — http://localhost:5173/
   - "Vol. 01 — Public Interest Monitor" 메타
   - "Ghost bidders in our school cafeterias" 헤드
   - 4분할 stats — 의심 클러스터 12, 관찰 사업자 N명 등
2. (00:15) **고위험 클러스터 클릭** — `BSN-2026-0001 동래구 동일주소 5인`
   - 카드 expand → 멤버 5명 / 관계망 / 24개월 타임라인
3. (00:30) **상세 페이지 이동** — `상세 보기` 버튼
   - RiskGauge 92점 / 신호 4분할 / 멤버 테이블
   - 시계열 차트 + 학교 분포 지도
4. (00:45) **매칭 판례** 카드 클릭 → `2023고단1234 (가상)` 상세
   - 가족 명의 4개 사업자, 동일 IP 응찰 패턴
5. (00:55) **정보공개청구 양식 만들기** — 액션 바
   - 다이얼로그 → ".doc 다운로드" 클릭

## 5분 데모 (기자·연구자용)

1. (00:00–00:30) **메인 대시보드** — 위와 동일
2. (00:30–01:30) **사이드바 필터링**
   - 고위험만 → 4건 / 동래구 → 3건 / "동일주소" 신호 → 다중 선택
   - 검색 `9991` → 동래구 클러스터만
3. (01:30–02:30) **클러스터 상세** — `BSN-2026-0001`
   - RiskGauge / 4 신호 패널 / 멤버 테이블 (행 클릭 → drawer)
   - 시계열 차트 hover 툴팁
4. (02:30–03:30) **학교별 보기** — `/schools`
   - 부산 지도 마커 색상 (의심 빈도)
   - `PSN-001` 클릭 → 우측 SchoolSummary + SafetyGauge
   - 학교 상세 → TOP 10 응찰자 + 응찰 이력
5. (03:30–04:30) **판례 라이브러리** — `/cases`
   - 패턴 필터 (동일주소·가족명의)
   - 판례 상세 → 매칭 클러스터 자동 추천
6. (04:30–05:00) **방법론 페이지** — `/methodology`
   - 데이터 소스 / 신호 / 클러스터링 / 한계 / 거버넌스
